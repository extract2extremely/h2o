## 💰 Currency Formatting Examples

### Standard Amounts
```javascript
// Small size (0.875rem)
CurrencyFormatter.format(50000, 'small');

// Medium size (1.125rem) - Default
CurrencyFormatter.format(50000, 'medium');

// Large size (1.5rem)
CurrencyFormatter.format(50000, 'large');
```

### Styled Amounts
```javascript
// Success style (Green)
CurrencyFormatter.format(100000, 'large', 'success');

// Warning style (Orange)
CurrencyFormatter.format(50000, 'medium', 'warning');

// Danger style (Red)
CurrencyFormatter.format(25000, 'small', 'danger');

// Info style (Blue)
CurrencyFormatter.format(75000, 'large', 'info');
```

### Special Formats

#### Badge Style
```javascript
CurrencyFormatter.formatBadge(150000);
// Output: Animated gradient badge with taka icon
```

#### Highlighted Amount
```javascript
CurrencyFormatter.formatHighlight(500000, 'Total Collection');
// Output: Highlighted box with label and amount
```

#### Summary Display (Large)
```javascript
CurrencyFormatter.formatSummary(1250000, 'Monthly Revenue');
// Output: Large summary card format
```

#### Table Cell
```javascript
CurrencyFormatter.formatTableCell(75000, 'success');
// Output: Compact format suitable for tables
```

#### Clickable (Copy to Clipboard)
```javascript
CurrencyFormatter.formatClickable(150000, 'Click to copy');
// Output: Clickable amount that copies to clipboard
```

#### Range Display
```javascript
CurrencyFormatter.formatRange(10000, 100000);
// Output: ৳10,000 - ৳100,000
```

#### Comparison with Change
```javascript
CurrencyFormatter.formatComparison(150000, 120000);
// Output: Shows current amount, change, and percentage
```

## ✅ Best Practices

### 1. Always Use Formatter for Consistency
```javascript
// ✓ Good
document.getElementById('amount').innerHTML = CurrencyFormatter.format(50000);

// ✗ Avoid
document.getElementById('amount').innerHTML = '<span>50,000</span>';
```

### 2. Use Appropriate Size for Context
```javascript
// Dashboard: Large
CurrencyFormatter.format(amount, 'large');

// Tables: Small
CurrencyFormatter.formatTableCell(amount);

// Summaries: Large
CurrencyFormatter.formatSummary(amount, label);
```

### 3. Semantic Color Usage
```javascript
// Success = Income/Collection
CurrencyFormatter.format(income, 'large', 'success');

// Danger = Expenses/Debt
CurrencyFormatter.format(debt, 'large', 'danger');

// Warning = At-risk/Pending
CurrencyFormatter.format(pending, 'large', 'warning');
```

### 4. PDF Generation
Always use PDFGenerator class for consistent professional output:
```javascript
const pdf = new PDFGenerator();
pdf.generateReport(title, data, options);
```

## 🐛 Troubleshooting

### Issue: Taka icon not displaying
**Solution:**
- Verify Font Awesome is loaded: `console.log(window.FontAwesome)`
- Check browser console for errors
- Ensure `fa-solid` class is present
- Confirm webfont files are accessible

### Issue: Currency formatter not working
**Solution:**
- Verify script loaded: `console.log(window.CurrencyFormatter)`
- Check that `currency-formatter.js` is included before use
- Review browser console for JavaScript errors

### Issue: PDF generation errors
**Solution:**
- Ensure jsPDF library is loaded: `console.log(window.jsPDF)`
- Verify data format matches expected structure
- Check PDF generator logs in console
