## 💡 Usage Tips

### For Digital Viewing
1. **Open in Browser**: Reports open in browser overlay
2. **Zoom**: Use Ctrl+Scroll or browser zoom controls
3. **Print Preview**: Use browser print preview (Ctrl+P)
4. **Export**: Save as PDF via print dialog

1. Click "Print Statement" button
2. Adjust printer settings as needed
3. Select "Save as PDF" for digital archiving
4. Or print directly to physical paper

### For PDF Download
1. Use "Download PDF" button for direct file
2. PDF saves to Downloads folder
3. Professional formatting maintained
4. Ready to email or archive

## 📄 Technical Details

### File Location
- Report styling: `/js/ui.js` (generateSavingsReport method)
- PDF generation: `/js/ui.js` (downloadPDF method)
- Library support: `/lib/jspdf/` (PDF engine)

### Libraries Used
- **jsPDF 2.5.1**: PDF generation engine
- **html2canvas 1.4.1**: HTML to canvas conversion
- **jsPDF AutoTable**: Professional table rendering
