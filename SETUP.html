⚙️ FinCollect - Setup Required

Your application needs to be run on a local web server

🚨 Why Am I Seeing This?
You're opening the file directly (file:// protocol), which causes browser security restrictions. FinCollect needs to run on a local web server (HTTP) to work properly.
✅ Quick Setup
🪟 Windows (Easiest)
Double-click the start-server.bat file in your FinCollect folder
The browser will open automatically at http://localhost:8000
🍎 macOS / 🐧 Linux
Open Terminal and run:
cd /path/to/Gravity
python3 server.py
Then open: http://localhost:8000
🔧 Any OS - Python One-Liner
python -m http.server 8000 --directory /path/to/Gravity
Then open: http://localhost:8000
📖 Step-by-Step Setup
  1. Locate your FinCollect folder
    Default: H:\Gravity (Windows) or ~/Gravity (Mac/Linux)
  2. Choose your method:
    • Windows: Double-click start-server.bat
    • Python: Run server.py or python -m http.server
    • VS Code: Use Live Server extension
  3. Open your browser to:
    http://localhost:8000
  4. ✅ Done! App runs without errors

🆘 Troubleshooting
Python not found?
Install Python from https://www.python.org/downloads/
Make sure to check "Add Python to PATH" during installation
Port 8000 already in use?
Use a different port: python -m http.server 8080
Still having issues?
Read: SERVER_SETUP_GUIDE.md in your FinCollect folder
